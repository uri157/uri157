# 🧠 Key Projects

### Reservation Platform
Multi-tenant reservation system with a **policy-driven booking engine**, real-time administrator coordination via WebSockets, and production deployment workflows.

### Exchange Simulator
Deterministic **Binance-like exchange simulator** designed for strategy backtesting on historical market data.

### compose-vps-deploy
Deterministic **Docker Compose deployment framework** for VPS environments with health checks, migrations and safe rollouts.

### pgbackrest-compose-ops
Operational toolkit for **safe PostgreSQL backup and restore workflows** using pgBackRest in Docker Compose environments.

### Trading Chart Platform
Real-time market data visualization platform built with a **C++ backend API**, DuckDB time-series storage, and a **Next.js trading chart interface**.
